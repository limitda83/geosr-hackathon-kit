# Before / After — 수작업 → 에이전트화

## 우리가 자동화한 반복 수작업
지진해일 수치모델(ADCIRC 계열) 수행을 위한 10단계 전처리·실행 준비 과정. 작업자가 각 단계 폴더(`01_Deform_Plane` ~ `10_SLR_RUN`)에 직접 들어가 파일·코드·파라미터를 수동 수정하고 순차 실행하던 반복 작업을, 웹에서 파라미터만 설정하면 자동으로 전 단계를 수행/산출하도록 자동화.

## Before (기존 수작업 절차)
1. 시나리오(지진원/단층 파라미터, 해수면 상승 SSP 시나리오 등)마다 10개 단계 폴더를 순서대로 연다.
2. 각 단계 Python 스크립트(`Deform_plane.py`, `Tsu_xyz.py`, `01_Fort14.py`, `MSL_to_AHHW_*.py`, `Fort13_manning.py`, `Transport.py`, `SLR_Scenario.py`, `fort14_Create.py`, `fort14_Change_Elevation.py`, `Tsunami_SLR_RLT_CASE.py` 등)를 열어 입력 경로·케이스 번호·파라미터를 손으로 수정한다(스크립트마다 `_2/_3/_4` 케이스별 사본을 따로 둘 정도로 변형이 잦음).
3. 단계마다 수동 실행 → 산출물 확인 → 다음 단계 입력으로 수동 연결. 여러 시나리오를 돌리려면 이 과정을 시나리오 수만큼 전부 반복.
4. 결과는 단계 폴더 곳곳에 흩어져 있어 시나리오 간 비교가 번거로움.

## After (에이전트화된 절차)
1. 웹 플랫폼 좌측에서 시나리오 파라미터를 설정(또는 에이전트 채팅으로 셋팅값 지정).
2. "실행"을 누르면 파라미터가 단계별 설정값으로 변환되어 10단계가 자동 순차 수행(데모는 기존 결과/Mock으로 흐름 시연; 에이전트는 셋팅만 하고 모델을 직접 실행하지는 않음).
3. 여러 시나리오를 큐로 자동 수행 → 우측 3D 뷰에서 부산권(마린시티·해운대) 침수 결과를 비교.
